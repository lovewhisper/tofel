很多同学因为练习方法不对，没有练到位，导致能力和分数提不上去！


所以在开始之前，我们有必要认真了解下：托福阅读应该如何训练才到位，才能真正提高能力和分数！



提高能力冲击高（满）分5步做题法总结：



托福阅读分数的提高有赖于


1. 单词语法的语言层面不断熟练和深入掌握（练语感）


2. 文章的结构层面进行训练（练题型和逻辑思维）


3. 阅读速度层面进行刻意训练（练速度）


4. 注意力层面训练（练注意力与续航能力）


5. 考试心态和发挥训练 （练能力与分数的转化）


所以平日里对待练习材料（托福阅读真题文章）时，也要从这几个层面下手去训练。


技巧的掌握可以通过Week1的讲解和强化训练完成，其他的层面可以通过Week2/3的文章练习和讲解搞定。


那么文章练习的最关键的两个目的：


1. 读完文章后，对文章涉及的语言知识（词汇/句子结构）会有更新和更准确的理解和认知。


2. 对文章涉及的语言知识（词汇/句子结构），有更快更通顺流畅的反应。


就两个字： 准 + 快


所以希望大家（尤其是阅读能力一般，给够时间是能读懂，得高分，但是就是考试时阅读速度不够，时间不足）认真做好的就是重读训练和快速重读训练（直到达标）


在解决了理解层面的问题后训练自己大脑对文章信息处理的速度，反应敏锐度，通过重复去提高这类文章用词造句布局结构的熟练度，扩容自己的STM（short-term memory 短时记忆）。


解决了信息处理的负担，才能匀出更多思绪去做逻辑变换和语义对应，才能真正将Week1要学的题型技巧发挥出来！
